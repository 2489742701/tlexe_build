# UI快速开发工具 - 项目状态手册

## 📋 项目概述

**项目名称**：UI快速开发工具（窗口编辑器）
**项目类型**：PyQt6桌面应用程序
**主要功能**：可视化设计桌面应用程序界面，支持拖拽式设计、属性编辑、行为配置等

---

## 🏗️ 项目架构

### 目录结构
```
测试项目-窗口编辑器/
├── main.py                 # 程序入口
├── models/                 # 数据模型层
│   ├── __init__.py
│   ├── base.py            # 基础模型类
│   └── components.py      # 组件模型类
├── views/                  # 视图层
│   ├── __init__.py
│   ├── main_window.py     # 主窗口
│   ├── canvas.py          # 画布视图
│   ├── property_panel.py  # 属性面板
│   ├── welcome_page.py    # 欢迎页面
│   ├── tree_view.py       # 树形视图
│   └── logic_tree.py      # 逻辑树
├── controllers/            # 控制器层
│   ├── __init__.py
│   └── project_controller.py  # 项目控制器
├── runtime/                # 运行时模块
│   ├── __init__.py
│   ├── runner.py          # 运行时启动器
│   └── action_executor.py # 动作执行器
├── templates/              # 模板模块
│   ├── __init__.py
│   └── test_template.py   # 测试模板
└── utils/                  # 工具模块
    ├── __init__.py
    ├── settings.py        # 设置管理
    └── undo_manager.py    # 撤销管理器
```

### 2. 测试模板

## 📝 代码规范

### 1. 文件编码
- 所有Python文件使用UTF-8编码
- 文件开头有BOM标记（\ufeff）

### 2. 注释规范
- 类和函数使用Google风格文档字符串
- 重要代码块添加中文注释
- 不添加不必要的注释

### 3. 命名规范
- 类名：大驼峰（PascalCase）
- 函数名：小写+下划线（snake_case）
- 变量名：小写+下划线（snake_case）
- 私有属性：下划线开头（_private_attr）

### 4. 信号命名
- 使用过去式（moved, selected, resized）
- 使用pyqtSignal装饰器
