## 项目概述

傻瓜桌面开发工具是一个简单易用的桌面应用程序可视化开发工具，采用 **Galgame 模式** 的界面流转设计，让每个人都能轻松创建桌面应用程序。

### 核心理念

- **Galgame 模式**: 界面流转采用视觉小说式的分支逻辑，按钮可触发新窗口/事件
- **所见即所得**: 拖拽式组件设计，实时预览
- **傻瓜化操作**: 无需编程基础，通过属性面板配置行为

## 架构设计

```

├── main.py                 # 程序入口
```

### Galgame 模式数据流

```
主程序窗口 (WindowType.MAIN)
    │
    ├── 按钮 A [驱动型] ──────────────┐
    │       │                        │
    │       └── target_window_id ────┼──► 事件窗口 A
    │                                │       │
    ├── 按钮 B [驱动型] ──────────────┤       ├── 文本框
    │       │                        │       ├── 输入框
    │       └── target_window_id ────┼──► 事件窗口 B
    │                                │       │
    └── 按钮 C [关闭程序]             │       └── 按钮 [返回]
                                     │
                                     └──► 事件窗口 C
```

## 模块说明

### 1. 数据模型层 (models/)

#### ComponentModel (base.py)
组件基类，所有组件的父类。

```python
class ComponentModel(QObject):
    # 信号
    data_changed = pyqtSignal()          # 数据变化
    position_changed = pyqtSignal(int, int)  # 位置变化
    size_changed = pyqtSignal(int, int)      # 大小变化
    style_changed = pyqtSignal()             # 样式变化
    
    # 属性
    id: str           # 唯一标识符
    type: str         # 组件类型
    name: str         # 组件名称
    x, y: int         # 坐标
    width, height: int # 大小
    text: str         # 显示文本
    style: StyleConfig # 样式配置
    action: ActionConfig # 行为配置
```

#### WindowModel (window.py)
窗口/事件模型，支持主程序和事件两种类型。

```python
class WindowModel(QObject):
    # 属性
    id: str              # 窗口ID
    name: str            # 窗口名称
    window_type: WindowType  # MAIN 或 EVENT
    width, height: int   # 窗口大小
    components: List[str] # 组件ID列表
    trigger_button_id: str # 触发按钮ID (事件窗口)
```

#### ProjectModel (base.py)
项目模型，管理整个工程数据。

```python
class ProjectModel(QObject):
    # 信号
    component_added = pyqtSignal(str)
    window_added = pyqtSignal(str)
    current_window_changed = pyqtSignal(str)
    
    # 方法
    add_component(comp, window_id)  # 添加组件
    create_event_for_button(button_id)  # 创建事件
    save_to_file(path)  # 保存项目
    load_from_file(path)  # 加载项目
```

### 2. 视图层 (views/)

#### DesignerView (canvas.py)
设计器画布视图，支持缩放、平移、组件管理。

```python
class DesignerView(QGraphicsView):
    # 方法
    add_component_item(model) -> ComponentGraphicsItem
    remove_component_item(comp_id)
    select_item(comp_id)
    zoom_in() / zoom_out() / zoom_reset()
    get_desktop_size() -> (width, height)
```

#### ComponentGraphicsItem (canvas.py)
画布上的组件图形项，支持拖拽、选中、调整大小。

```python
class ComponentGraphicsItem(QGraphicsObject):
    # 信号
    moved = pyqtSignal(str, int, int)     # (id, x, y)
    selected = pyqtSignal(str)            # (id)
    resized = pyqtSignal(str, int, int)   # (id, w, h)
    
    # 常量
    HANDLE_SIZE = 8   # 缩放手柄大小
    MIN_SIZE = 20     # 最小尺寸
```

#### PropertyPanel (property_panel.py)
属性面板，显示和编辑选中组件的属性。

```python
class PropertyPanel(QWidget):
    # 信号
    property_changed = pyqtSignal(str, str, object)
    create_event_requested = pyqtSignal(str)  # 创建事件
    goto_event_requested = pyqtSignal(str)    # 跳转事件
    
    # 方法
    set_component(model)  # 设置当前组件
    clear()  # 清空面板
```

### 3. 控制器层 (controllers/)

#### ProjectController (project_controller.py)
项目控制器，协调模型和视图。

```python
class ProjectController(QObject):
    # 初始化
    __init__(window, model)
    
    # 项目操作
    _on_new_project()
    _on_open_project(file_path)
    _on_save_project()
    _on_run_project()
    
    # 组件操作
    _on_add_component(comp_type, parent_id)
    _on_delete_component(comp_id)
    _select_component(comp_id)
    
    # 窗口操作
    _switch_to_window(window_id)
    _create_event_for_button(button_id)
```

### 4. 运行时 (runtime/)

#### ActionExecutor (action_executor.py)
行为执行器，执行配置的动作。

```python
class ActionExecutor:
    def execute(action_config: dict):
        # 支持的动作类型
        - open_file: 打开文件
        - run_command: 运行命令
        - open_url: 打开网址
        - close_window: 关闭窗口
        - show_message: 显示消息
        - python_code: 执行Python代码
```

#### Runner (runner.py)
运行时启动器，加载项目并启动应用。

```python
class Runner:
    @staticmethod

    @staticmethod
    def run_from_data(project_data: dict)

class RuntimeManager:
    def start()  # 启动主窗口

```

## 使用指南

### 启动程序

```bash
python main.py
```

### 基本操作

1. **添加组件**: 点击工具栏的"+按钮"、"+标签"等按钮
2. **移动组件**: 在画布上拖拽组件
3. **调整大小**: 选中组件后拖拽边缘手柄
4. **编辑属性**: 在右侧属性面板修改组件属性
5. **创建事件**: 选中按钮，在属性面板勾选"创建事件分支"
6. **切换窗口**: 在左侧逻辑树点击不同窗口节点
7. **保存项目**: 文件 → 保存 (Ctrl+S)
8. **运行项目**: 按 F5 或点击运行按钮

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建项目 |
