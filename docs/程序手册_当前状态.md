## 项目概述

傻瓜桌面开发工具是一个基于 PyQt6 的可视化桌面应用开发工具，采用 **MVC 架构** 和 **Galgame 模式** 的界面流转设计。

### 核心理念

- **Galgame 模式**: 按钮可触发新窗口/事件，形成分支结构
- **所见即所得**: 拖拽式设计，实时预览
- **傻瓜化操作**: 无需编程基础

## 核心模块

### 1. 数据模型层 (models/)

#### ComponentModel (base.py)
组件基类，所有组件的父类。

```python
class ComponentModel(QObject):
    # 信号
    data_changed = pyqtSignal()              # 数据变化
    position_changed = pyqtSignal(int, int)  # 位置变化 (x, y)
    size_changed = pyqtSignal(int, int)      # 大小变化 (w, h)
    style_changed = pyqtSignal()             # 样式变化
    action_changed = pyqtSignal()            # 行为变化
    
    # 核心属性
    id: str              # 唯一标识符 (UUID前8位)
    type: str            # 组件类型 (button/label/input/container)
    name: str            # 组件名称
    x: int               # 横坐标
    y: int               # 纵坐标
    width: int           # 宽度
    height: int          # 高度
    text: str            # 显示文本
    parent_id: str       # 父组件ID
    style: StyleConfig   # 样式配置
    action: ActionConfig # 行为配置
    visible: bool        # 是否可见
    enabled: bool        # 是否可用
```

#### ButtonModel (components.py)
按钮组件，支持分支跳转。

```python
class ButtonModel(ComponentModel):
    # 特有属性
    is_default: bool         # 是否为默认按钮(回车触发)
    is_cancel: bool          # 是否为取消按钮(ESC触发)
    target_window_id: str    # 目标窗口ID
    branch_name: str         # 分支名称
    
    @property
    def has_branch(self) -> bool:
        """是否有分支(根据target_window_id判断)"""
        return bool(self.target_window_id)
```

class LabelModel(ComponentModel):

#### ProjectModel (base.py)
项目模型，管理整个工程。

```python
class ProjectModel(QObject):
    # 信号
    component_added = pyqtSignal(str)        # 组件添加 (comp_id)
    component_removed = pyqtSignal(str)      # 组件移除 (comp_id)
    component_changed = pyqtSignal(str)      # 组件变化 (comp_id)
    window_added = pyqtSignal(str)           # 窗口添加 (window_id)
    window_removed = pyqtSignal(str)         # 窗口移除 (window_id)
    window_changed = pyqtSignal(str)         # 窗口变化 (window_id)
    current_window_changed = pyqtSignal(str) # 当前窗口变化
    project_loaded = pyqtSignal()            # 项目加载完成
    project_saved = pyqtSignal()             # 项目保存完成
    
    # 属性
    name: str                    # 项目名称
    components: Dict[str, ComponentModel]  # 组件字典
    windows: Dict[str, WindowModel]        # 窗口字典
    main_window_id: str          # 主窗口ID
    current_window_id: str       # 当前窗口ID
    
    # 核心方法
    add_component(comp, window_id) -> str
    remove_component(comp_id) -> bool
    get_component(comp_id) -> ComponentModel
    create_event_for_button(button_id) -> str
    save_to_file(path) -> bool
    load_from_file(path) -> bool
```

## 使用说明

### 启动程序

```bash
python main.py
```

### 基本操作

1. **添加组件**: 点击工具栏的"+按钮"、"+标签"等
2. **移动组件**: 在画布上拖拽
3. **调整大小**: 选中后拖拽边缘手柄
4. **编辑属性**: 在右侧属性面板修改
5. **创建事件**: 选中按钮 → 属性面板 → "创建事件分支"
6. **切换窗口**: 点击左侧逻辑树的窗口节点
7. **缩放画布**: Alt+滚轮 或 Ctrl++/-
8. **缩放逻辑树**: Ctrl+滚轮
9. **保存项目**: Ctrl+S
10. **运行项目**: F5

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建项目 |
| Ctrl+O | 打开项目 |
| Ctrl+S | 保存项目 |
| Ctrl+Shift+S | 另存为 |
| Ctrl++ | 放大 |
| Ctrl+- | 缩小 |
| Ctrl+0 | 重置缩放 |
| F5 | 运行项目 |
| Delete | 删除选中组件 |
| Alt+滚轮 | 画布缩放 |
| Ctrl+滚轮 | 逻辑树缩放 |
| 中键拖拽 | 画布平移 |

## 注意事项

1. **容器居中**: 新添加的容器会自动显示在画布中央
2. **预览模式**: 运行时使用容器作为主窗口，不显示花布背景
3. **逻辑树缩放**: 支持 Ctrl+滚轮缩放，方便查看复杂分支
4. **分支跳转**: 按钮创建事件后，在逻辑树中会显示分支箭头
5. **项目文件**: 使用 .fool 格式，本质是 JSON 文件
