## 核心类说明

### 1. 模型层 (models/)

#### ComponentModel (base.py)
- 所有组件的基类
- 属性: id, type, name, x, y, width, height, text, parent_id, style
- 信号: data_changed, style_changed
- 方法: to_dict(), set_size(width, height)

#### ButtonModel (components.py)
- 继承自 ComponentModel
- 特有属性: is_default, is_cancel, target_window_id, branch_name
- 默认样式: 蓝色背景 (#4a90d9), 白色文字

#### LabelModel (components.py)
- 继承自 ComponentModel
- 特有属性: alignment, word_wrap

#### InputModel (components.py)
- 继承自 ComponentModel
- 特有属性: placeholder, is_password, is_multiline, max_length

#### ContainerModel (components.py)
- 继承自 ComponentModel
- 特有属性: layout, padding, spacing, container_type
- container_type: 'embedded'(嵌入式) 或 'popup'(弹窗)

#### ProjectModel (base.py)
- 项目模型，管理所有窗口和组件
- 属性: name, file_path, windows, components
- 信号: component_added, component_removed, window_added, window_removed
- 支持 Galgame 模式的多窗口管理

### 2. 视图层 (views/)

#### DesignerView (canvas.py)
- 主画布视图
- 功能: 滚轮缩放、中键平移、桌面模拟
- 默认桌面: 1920x1080
- 缩放范围: 0.1 - 4.0
- 信号: zoom_changed

#### ComponentGraphicsItem (canvas.py)
- 画布上的组件图形项
- 支持拖拽移动、选中高亮、8个方向调整大小
- 手柄布局:
  ```
    0 ----- 1 ----- 2
    |               |
    7               3
    |               |
    6 ----- 5 ----- 4
  ```
- 信号: moved, selected, resized

#### PropertyPanel (property_panel.py)
- 属性编辑面板
- 分组: 基础属性、位置大小、样式、类型特定属性
- 信号: property_changed, action_config_requested

#### LogicTreeView (logic_tree.py)
- 逻辑树视图 (Galgame模式)
- 显示: 主程序 -> 按钮 -> 事件窗口 的分支结构
- 支持 Ctrl+滚轮 缩放

### 3. 控制器层 (controllers/)

#### ProjectController (project_controller.py)
- 协调模型和视图
- 处理: 新建、打开、保存、导出、运行
- 管理: 组件添加/删除/重命名、撤销/重做、剪切/复制/粘贴

## 已修复的问题

### 1. 画布缩放问题
- **问题**: 缩放锚点错误，导致缩放时位置偏移
- **修复**: `setTransformationAnchor(AnchorViewCenter)`
- **文件**: canvas.py

### 2. 组件调整大小崩溃
- **问题**: 向上/向左调整大小时崩溃
- **原因**: 位置计算逻辑错误，使用了变化后的值
- **修复**: 添加 `_resize_start_x`, `_resize_start_y` 保存初始位置
- **文件**: canvas.py

### 3. 样式属性更新不生效
- **问题**: 修改样式属性后画布不重绘
- **修复**: 在属性修改后发射 `data_changed.emit()`
- **文件**: property_panel.py

### 4. 组件树添加组件不工作
- **问题**: `_on_add_component` 方法只有 `pass`
- **修复**: 添加 `add_component_requested` 信号并连接到控制器
- **文件**: component_tree.py

### 5. BOM字符问题
- **问题**: 多个文件包含 UTF-8 BOM 字符导致语法错误
- **修复**: 重写文件去除 BOM
- **文件**: main_window.py, logic_tree.py 等

## 代码规范

### 1. 信号命名
- 使用过去式: `moved`, `selected`, `resized`
- 请求类信号: `xxx_requested`

### 2. 属性定义
- 私有属性: `_property_name`
- 使用 @property 装饰器
- setter 中发射 `data_changed.emit()`

### 3. 类型提示
- 使用 Python 类型提示: `Optional[str]`, `list[ComponentModel]`
- 参数类型和返回值类型都要标注

### 4. 文档字符串
- 使用三引号文档字符串
- 包含功能说明、参数说明、返回值说明

## 常见问题排查

### 1. 程序无法启动
- 检查 Python 版本 (需要 3.10+)
- 检查 PyQt6 是否安装: `pip install PyQt6`
- 检查文件是否有 BOM 字符

### 2. 组件无法拖动
- 检查 `ItemIsMovable` 标志是否设置
- 检查是否有事件拦截

### 3. 属性修改不生效
- 检查信号是否正确连接
- 检查 `_updating` 标志是否阻止了更新

### 4. 调整大小崩溃
- 检查 `_resize_start_x`, `_resize_start_y` 是否正确初始化
- 检查尺寸是否小于 MIN_SIZE
