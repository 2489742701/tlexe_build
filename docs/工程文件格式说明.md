# 工程文件格式说明 (.itexe)

> 本文档说明傻瓜桌面开发工具的项目文件格式。

---

## 概述

项目文件使用 `.itexe` 后缀，采用 JSON 格式存储。包含项目的基本信息、窗口定义和组件列表。

---

## 文件结构

```json
{
  "name": "项目名称",
  "main_window_id": "主窗口ID",
  "current_window_id": "当前窗口ID",
  "windows": [...],
  "components": [...]
}
```

---

## 顶层字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 项目名称 |
| `main_window_id` | string | 主窗口的ID |
| `current_window_id` | string | 当前编辑的窗口ID |
| `windows` | array | 窗口列表 |
| `components` | array | 组件列表 |

---

## 窗口定义 (windows)

```json
{
  "id": "window001",
  "name": "窗口名称",
  "window_type": "main",
  "width": 800,
  "height": 600,
  "title": "窗口标题",
  "components": ["comp001", "comp002"]
}
```

### 窗口字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 窗口唯一标识符 |
| `name` | string | 窗口名称（显示用） |
| `window_type` | string | 窗口类型：`main`（主窗口）或 `event`（事件窗口） |
| `width` | int | 窗口宽度 |
| `height` | int | 窗口高度 |
| `title` | string | 窗口标题栏文本 |
| `components` | array | 该窗口包含的组件ID列表 |

### 窗口类型

- **main**: 主程序窗口，程序启动时显示
- **event**: 事件窗口，由按钮触发打开

## 注意事项

1. **ID唯一性**: 所有窗口ID和组件ID必须唯一
2. **编码格式**: 文件必须使用 UTF-8 编码保存
3. **颜色格式**: 颜色使用十六进制格式，如 `#ff0000`
4. **透明背景**: 使用 `transparent` 表示透明背景
5. **父容器关系**: `parent_id` 必须指向一个存在的容器组件
