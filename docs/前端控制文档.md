## 画布模块

### 核心类

#### 1. ComponentGraphicsItem (组件图形项)

**功能**: 画布上的可视化组件，支持拖拽、选中、调整大小。

**信号**:

| 信号名        | 参数                                 | 说明        |
| ---------- | ---------------------------------- | --------- |
| `moved`    | (id: str, x: int, y: int)          | 组件移动时发射   |
| `selected` | (id: str)                          | 组件被选中时发射  |
| `resized`  | (id: str, width: int, height: int) | 组件大小改变时发射 |

**关键属性**:

| 属性              | 类型   | 说明            |
| --------------- | ---- | ------------- |
| `HANDLE_SIZE`   | int  | 调整手柄大小，默认8    |
| `MIN_SIZE`      | int  | 最小组件尺寸，默认20   |
| `_grid_size`    | int  | 网格大小，默认10     |
| `_snap_to_grid` | bool | 是否吸附网格，默认True |

**关键方法**:

```python
# 获取组件ID
item.component_id  # 返回 str

# 获取关联的模型
item.model  # 返回 ComponentModel

# 获取边界矩形
item.boundingRect()  # 返回 QRectF

# 判断是否选中
item.isSelected()  # 返回 bool
```

**拖拽流程**:

1. 用户点击组件 → `mousePressEvent` 触发
2. 检查是否点击调整手柄 → `_get_handle_at()`
3. 如果是手柄 → 进入调整大小模式 (`_resizing = True`)
4. 如果不是手柄 → 进入拖拽模式 (Qt内置)
5. 移动时 → `mouseMoveEvent` → `_do_resize()` 或 Qt自动处理
6. 释放时 → `mouseReleaseEvent` → 发射 `moved` 或 `resized` 信号

**调整手柄位置** (8个点):

```
  0 ----- 1 ----- 2
  |               |
  7               3
  |               |
  6 ----- 5 ----- 4
```

**鼠标光标映射**:

| 手柄   | 光标              | 方向    |
| ---- | --------------- | ----- |
| 0, 4 | SizeFDiagCursor | 对角线 ↘ |
| 1, 5 | SizeVerCursor   | 垂直 ↕  |
| 2, 6 | SizeBDiagCursor | 对角线 ↙ |
| 3, 7 | SizeHorCursor   | 水平 ↔  |

#### 2. DesignerScene (设计器场景)

**功能**: 管理画布背景、桌面区域、网格显示。

**关键属性**:

| 属性                      | 默认值     | 说明      |
| ----------------------- | ------- | ------- |
| `_desktop_width`        | 1920    | 桌面宽度    |
| `_desktop_height`       | 1080    | 桌面高度    |
| `_grid_size`            | 10      | 网格大小    |
| `_grid_color`           | #e0e0e0 | 网格颜色    |
| `_desktop_border_color` | #333333 | 桌面边框颜色  |
| `_outside_color`        | #1a1a1a | 桌面外区域颜色 |
| `_desktop_bg_color`     | #f5f5f5 | 桌面背景颜色  |

**关键方法**:

```python
# 获取桌面矩形
scene.desktop_rect  # 返回 QRectF(0, 0, 1920, 1080)

# 设置桌面尺寸
scene.set_desktop_size(width: int, height: int)
```

**绘制流程** (`drawBackground`):

1. 填充整个场景为深色 (`_outside_color`)
2. 填充桌面区域为浅色 (`_desktop_bg_color`)
3. 绘制桌面边框 (3像素黑边)
4. 绘制网格线 (如果 `_show_grid = True`)

#### 3. DesignerView (设计器视图)

**功能**: 主画布视图，支持缩放、平移、组件管理。

**关键属性**:

| 属性                  | 默认值   | 说明          |
| ------------------- | ----- | ----------- |
| `_zoom_factor`      | 1.0   | 当前缩放比例      |
| `_min_zoom`         | 0.1   | 最小缩放比例      |
| `_max_zoom`         | 4.0   | 最大缩放比例      |
| `_items`            | {}    | 组件ID到图形项的映射 |
| `_panning`          | False | 是否正在平移      |
| `_initial_fit_done` | False | 是否已完成初始适配   |

**关键方法**:

```python
# 添加组件
view.add_component_item(model: ComponentModel) -> ComponentGraphicsItem

# 移除组件
view.remove_component_item(comp_id: str)

# 获取组件项
view.get_item(comp_id: str) -> Optional[ComponentGraphicsItem]

# 选中组件
view.select_item(comp_id: str)

# 清除选择
view.clear_selection()

# 获取选中项
view.get_selected_items() -> list[ComponentGraphicsItem]

# 缩放控制
view.zoom_in()    # 放大 1.2 倍
view.zoom_out()   # 缩小 0.8 倍
view.zoom_reset() # 重置到适配视图

# 桌面尺寸
view.get_desktop_size() -> tuple[int, int]
view.set_desktop_size(width: int, height: int)
```

**事件处理**:

| 事件                  | 触发条件   | 行为      |
| ------------------- | ------ | ------- |
| `wheelEvent`        | 滚轮滚动   | 缩放视图    |
| `mousePressEvent`   | 中键按下   | 开始平移    |
| `mouseMoveEvent`    | 平移中    | 移动视图    |
| `mouseReleaseEvent` | 中键释放   | 结束平移    |
| `showEvent`         | 首次显示   | 适配桌面到视图 |
| `resizeEvent`       | 窗口大小改变 | 重新适配    |

**缩放锚点**:

- `setTransformationAnchor(AnchorViewCenter)` - 以视图中心为锚点缩放
- `setResizeAnchor(AnchorViewCenter)` - 窗口大小改变时以中心为锚点

## 属性面板模块

### PropertyPanel 类

**功能**: 显示和编辑选中组件的属性。

**信号**:

| 信号名                       | 参数                                | 说明       |
| ------------------------- | --------------------------------- | -------- |
| `property_changed`        | (comp\_id, property\_name, value) | 属性改变     |
| `action_config_requested` | (comp\_id)                        | 请求配置行为   |
| `create_event_requested`  | (button\_id)                      | 请求创建事件分支 |
| `goto_event_requested`    | (window\_id)                      | 请求跳转到事件  |

**关键方法**:

```python
# 设置当前组件
panel.set_component(model: Optional[ComponentModel])

# 清空面板
panel.clear()
```

**属性分组**:

1. **基础属性** (basic\_group)
   - ID (只读)
   - 类型 (只读)
   - 名称 (可编辑)
   - 文本 (可编辑)
2. **位置和大小** (geometry\_group)
   - X坐标 (0-10000)
   - Y坐标 (0-10000)
   - 宽度 (10-10000)
   - 高度 (10-10000)
3. **样式** (style\_group)
   - 背景颜色 (带颜色选择器)
   - 文本颜色 (带颜色选择器)
   - 边框颜色 (带颜色选择器)
   - 边框宽度 (0-20)
   - 圆角半径 (0-50)
   - 字体大小 (6-72)
   - 粗体 (复选框)
4. **类型特定属性**:
   - **ButtonModel**: is\_default, is\_cancel, 分支配置, 动作配置
   - **LabelModel**: alignment, word\_wrap
   - **InputModel**: placeholder, is\_password, is\_multiline, max\_length
   - **ContainerModel**: layout, padding, spacing

**属性更新流程**:

1. 用户修改控件值 → 触发信号
2. `_on_property_changed` 被调用
3. 更新模型属性
4. 发射 `property_changed` 信号
5. 模型发射 `data_changed` 信号
6. 画布重绘组件

## 组件树模块

### ComponentTreeView 类

**功能**: 显示组件层级结构，支持选择、重命名、删除。

**信号**:

| 信号名                        | 参数                       | 说明     |
| -------------------------- | ------------------------ | ------ |
| `component_selected`       | (comp\_id)               | 选中组件   |
| `component_double_clicked` | (comp\_id)               | 双击组件   |
| `delete_requested`         | (comp\_id)               | 请求删除   |
| `rename_requested`         | (comp\_id, new\_name)    | 请求重命名  |
| `add_component_requested`  | (comp\_type, parent\_id) | 请求添加组件 |

**关键方法**:

```python
# 设置项目模型
tree.set_project_model(model: ProjectModel)

# 选中指定组件
tree.select_component(comp_id: str)

# 清除选择
tree.clear_selection()

# 获取选中组件ID
tree.get_selected_component_id() -> Optional[str]
```

**树项显示格式**:

```
组件名称 (组件类型)
```

**上下文菜单**:

- 重命名
- 删除

**拖拽支持**:

- `setDragEnabled(True)` - 允许拖拽
- `setAcceptDrops(True)` - 允许放置
- `setDragDropMode(InternalMove)` - 内部移动

## 常见问题与解决方案

### 1. 拖拽时组件不移动

**可能原因**:

- `ItemIsMovable` 标志未设置
- 组件被锁定
- 事件被父控件拦截

**解决方案**:

```python
item.setFlag(QGraphicsObject.GraphicsItemFlag.ItemIsMovable, True)
item.setFlag(QGraphicsObject.GraphicsItemFlag.ItemSendsGeometryChanges, True)
```

### 2. 缩放后组件位置偏移

**可能原因**:

- 缩放锚点设置错误
- 缩放后未重新居中

**解决方案**:

```python
view.setTransformationAnchor(QGraphicsView.ViewportAnchor.AnchorViewCenter)
# 缩放后重新居中
view.centerOn(desktop_center_x, desktop_center_y)
```

### 3. 属性面板修改不生效

**可能原因**:

- `_updating` 标志阻止了更新
- 信号未正确连接
- 属性名格式错误

**解决方案**:

```python
# 确保样式属性修改后发射信号
setattr(model.style, style_prop, value)
model.style_changed.emit()
model.data_changed.emit()
```

### 4. 组件树不刷新

**可能原因**:

- 未连接项目模型信号
- 组件添加后未发射信号

**解决方案**:

```python
model.component_added.connect(tree._on_component_added)
model.component_removed.connect(tree._on_component_removed)
```

### 5. 调整大小时崩溃

**可能原因**:

- `_resize_start_rect` 未初始化
- 计算结果为负数

**解决方案**:

```python
# 在 mousePressEvent 中初始化
self._resize_start_rect = self.boundingRect()

# 确保尺寸不小于最小值
new_width = max(self.MIN_SIZE, rect.width() + delta.x())
```

### 6. 绘制时类型错误

**可能原因**:

- PyQt6 的绘图方法需要整数参数
- 传入了浮点数

**解决方案**:

```python
# 错误
painter.drawLine(x1, y1, x2, y2)  # 如果是浮点数会报错

# 正确
painter.drawLine(int(x1), int(y1), int(x2), int(y2))
```

## 快速参考

### 创建组件

```python
from models import create_component

button = create_component('button', x=100, y=100, text="确定")
label = create_component('label', x=50, y=50, text="标题")
input_box = create_component('input', x=100, y=150, placeholder="请输入")
container = create_component('container', x=0, y=0, width=400, height=300)
```

### 添加到画布

```python
item = view.add_component_item(button)
```

### 监听组件变化

```python
button.data_changed.connect(lambda: print("组件已更新"))
```

### 程序化移动组件

```python
button.x = 200
button.y = 200
# 会自动发射 data_changed 信号，画布会重绘
```
