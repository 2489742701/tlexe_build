# 傻瓜桌面开发工具

一个简单易用的桌面应用程序可视化开发工具。支持拖拽式设计、属性编辑、行为配置等功能。让每个人都能轻松创建桌面应用程序！

## 功能特性

### 核心功能
- **可视化设计**：拖拽式组件设计，所见即所得
- **组件系统**：支持按钮、标签、输入框、容器等多种组件
- **属性编辑**：完整的属性面板，实时预览修改效果
- **行为配置**：为组件配置点击行为（打开文件、运行命令、显示消息等）
- **项目保存/加载**：JSON格式项目文件，方便分享和版本控制
- **实时运行**：一键运行设计的应用程序

### 高级功能
- **网格对齐**：画布网格辅助对齐，支持10px网格
- **画布缩放**：Ctrl+滚轮缩放，支持25%-400%缩放范围
- **组件树管理**：左侧组件树，支持层级关系管理
- **撤销/重做**：基于QUndoCommand的撤销重做框架
- **样式定制**：支持颜色、字体、边框、圆角等样式设置

## 使用方法

### 启动程序
```bash
python main.py
```

### 基本操作
1. **添加组件**：点击工具栏的"+按钮"、"+标签"等按钮
2. **移动组件**：在画布上拖拽组件
3. **调整大小**：选中组件后拖拽边缘手柄
4. **编辑属性**：在右侧属性面板修改组件属性
5. **配置行为**：点击属性面板的"配置行为"按钮
6. **保存项目**：文件 → 保存（或Ctrl+S）
7. **运行项目**：按F5或点击运行按钮

### 快捷键
- **Ctrl+N**：新建项目
- **Ctrl+O**：打开项目
- **Ctrl+S**：保存项目
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做
- **Ctrl++**：放大
- **Ctrl+-**：缩小
- **Ctrl+0**：重置缩放
- **F5**：运行项目
- **Delete**：删除选中组件

## 技术栈

- **Python 3.8+**
- **PyQt6**：GUI框架
- **MVC架构**：模型-视图-控制器分离
- **Signal-Slot机制**：组件间通信

## 组件类型

### 基础组件
- **Button（按钮）**：可点击，支持配置点击行为
- **Label（标签）**：显示文本，支持对齐方式设置
- **Input（输入框）**：文本输入，支持密码模式、占位符
- **Container（容器）**：容器组件，支持布局管理

### 行为类型
- **打开文件**：使用系统默认程序打开指定文件
- **运行命令**：执行系统命令（非阻塞）
- **打开网址**：使用默认浏览器打开网址
- **关闭窗口**：关闭当前窗口
- **显示消息**：显示消息对话框
- **执行Python代码**：执行自定义Python代码

## 文件格式

项目文件使用JSON格式（.fool），包含：
- 项目元数据（名称、窗口大小、标题）
- 组件列表及其属性
- 样式配置
- 行为配置

示例：
```json
{
  "name": "我的应用",
  "width": 800,
  "height": 600,
  "title": "Hello World",
  "components": [
    {
      "id": "abc123",
      "type": "button",
      "name": "button_abc123",
      "x": 100,
      "y": 100,
      "width": 100,
      "height": 36,
      "text": "点击我",
      "style": {
        "background_color": "#4a90d9",
        "text_color": "#ffffff"
      },
      "action": {
        "action_type": "show_message",
        "params": {"param": "Hello World!"}
      }
    }
  ]
}
```
